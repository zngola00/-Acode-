كيفية تثبيت التطبيق Acode

لتثبيت Acode على هاتفك:

  1. افتح متجر Google Play على أندرويد أو App Store على iPhone
  2. ابحث عن "Acode" في شريط البحث
  3. اضغط على زر "تثبيت"
  4. انتظر حتى يكتمل التثبيت
  5. اضغط على "فتح" لبدء استخدام التطبيق
نصيحة: تأكد من أن لديك مساحة كافية على هاتفك قبل التثبيت.

كيفية تسمية الملف وحفظه

لإنشاء ملف جديد وحفظه في Acode:

  1. اضغط على زر القائمة ☰
  2. اختر "New File"
  3. أدخل اسم الملف مع الامتداد الصحيح (مثل index.html)
  4. اضغط على "موافق"
  5. اكتب محتوى الملف
  6. اضغط على أيقونة الحفظ 💾 أو اختر "Save" من القائمة
أمثلة على أسماء الملفات الصحيحة:
index.html
style.css
script.js

كيفية كتابة العنوان الأساسي

في HTML، نستخدم وسوم h1 إلى h6 للعناوين:

<h1>هذا عنوان رئيسي</h1>
<h2>هذا عنوان فرعي</h2>
<h3>عنوان أصغر</h3>

لجعل العنوان في المنتصف نستخدم CSS:

h1 {
  text-align: center;
  color: blue;
}

كيفية تغيير الألوان وحجم الخط

لتغيير الألوان في CSS:

body {
  color: #333; /* لون النص */
  background-color: #f8f9fa; /* لون الخلفية */
  font-size: 16px; /* حجم الخط */
}

يمكن استخدام الألوان بعدة طرق:

  • أسماء الألوان: red, blue, green
  • أكواد HEX: #ffffff (أبيض), #000000 (أسود)
  • RGB: rgb(255, 0, 0) (أحمر)

كيفية إضافة الصور والخلفيات

لإضافة صورة في HTML:

<img src="صورة.jpg" alt="وصف الصورة">

لإضافة خلفية في CSS:

body {
  background-image: url('خلفية.jpg');
  background-size: cover;
}

كيفية إنشاء أزرار تفاعلية

لإنشاء زر في HTML:

<button class="my-button">اضغط هنا</button>

لتنسيق الزر في CSS:

.my-button {
  padding: 10px 20px;
  background-color: #4CAF50;
  color: white;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

.my-button:hover {
  background-color: #45a049;
}

كيفية إضافة تأثيرات حركية

لإضافة حركة في CSS:

.animated-box {
  width: 100px;
  height: 100px;
  background: red;
  transition: all 0.3s ease;
}

.animated-box:hover {
  transform: rotate(45deg);
  background: blue;
}

لإنشاء حركة متكررة:

@keyframes example {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

.spinner {
  animation: example 2s linear infinite;
}

كيفية جعل الموقع متجاوباً

لجعل الموقع متجاوباً مع جميع أحجام الشاشات:

/* للهواتف */
@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}

/* للأجهزة اللوحية */
@media (min-width: 601px) and (max-width: 1024px) {
  .container {
    width: 90%;
  }
}
نصيحة: دائماً ابدأ بتصميم نسخة الهاتف أولاً (Mobile First Design).
